## 六、生成参数设置

### Midjourney V6 推荐参数
```
--ar 4:5        # 产品图
--ar 16:9       # Hero/课程
--ar 9:16       # 手机版
--ar 21:9       # 宽屏沉浸
--style raw     # 减少AI美化，更真实
--v 6.0         # 版本6
--s 250         # 风格化程度 (0-1000)
--q 2           # 质量最高
```

### Stable Diffusion 推荐设置
```
Model: realisticVisionV51
Sampler: DPM++ 2M Karras
Steps: 30-40
CFG Scale: 7-8
Resolution: 1024x1280 (4:5) 或 1216x832 (16:9)
```

### 后期处理建议
1. **统一调色**: 所有图片添加轻微暖调 (5500K)
2. **锐化**: 适度锐化突出食物纹理
3. **裁剪**: 确保构图符合品牌网格系统
4. **压缩**: WebP格式，质量85%
