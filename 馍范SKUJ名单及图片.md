# 馍范品牌视觉规范与SKU产品图生成指南

> **版本**：v2.0 - 品牌一致性版  
> **用途**：独立站全站产品卡片视觉统一  
> **核心要求**：11个SKU放在一起展示时必须视觉和谐，符合东方美学食育品牌调性

---

## 一、馍范品牌视觉系统（Visual Identity System）

### 1.1 色彩体系（Color Palette）

| 颜色角色 | 色值 | 使用场景 |
|---------|------|---------|
| **主色 - 薄荷青** | `#00B894` | 品牌强调、按钮、标签 |
| **辅助色 - 暖白** | `#F5F5F0` | 主背景、面团高光 |
| **中性色 - 原木** | `#D4A574` | 道具（蒸笼/砧板）、温暖感 |
| **点缀色 - 炭灰** | `#2D3436` | 文字、阴影、高级感 |
| **点缀色 - 浅青** | `#55EFC4` | 蒸汽、光晕、清新感 |

### 1.2 光影规范（Lighting System）

**统一光源设定**：
- **类型**：自然窗光（模拟早晨8-9点侧逆光）
- **方向**：45度侧逆（从左上方射入）
- **质感**：柔和漫射（纱帘柔化）+ 微暖调（5500K偏暖）
- **效果**：面团表面细腻光泽、蒸汽朦胧感、背景渐变虚化

**禁止**：直射硬光、闪光灯平光、冷白光（破坏温暖手作感）

### 1.3 构图体系（Composition Grid）

**独立站产品卡片标准比例**：`4:5`（竖版，适合手机浏览）

**统一构图法则**：
- **主体位置**：黄金分割点（画面右侧1/3处或正中偏下）
- **留白比例**：上方40%留白（放SKU标题）、下方20%留白（放价格/按钮）
- **景深控制**：f/2.8-f/4，主体清晰，背景虚化至品牌色系
- **视角体系**：
  - 引流款：45度俯视（亲切感）
  - 利润款：横截面特写（专业感）
  - 旗舰款：平视微仰（仪式感）

### 1.4 道具系统（Props Library）

**统一材质**：竹、陶瓷、棉麻、原木（东方美学四元素）

| 道具类型 | 具体物品 | 使用SKU |
|---------|---------|---------|
| **盛器** | 浅黄竹蒸笼（圆角） | 全系通用 |
| **背景** | 原木色桧木砧板 / 暖白亚麻布 | 引流/利润款 |
| **点缀** | 薄荷叶/麦穗/面粉筛（薄荷青铁艺） | 全系点缀 |
| **旗舰专属** | 青瓷盘/红宣纸/金箔 | 旗舰款节庆 |

## 六、Webflow展示一致性规范

### 6.1 产品卡片布局标准

```
┌─────────────────────────┐  ← 上方40%留白（Webflow叠加白色标题文字）
│                         │
│    [产品图片主体]        │  ← 右侧1/3黄金分割点
│    蒸汽/流心/切面细节     │
│                         │
├─────────────────────────┤  ← 下方20%留白（Webflow叠加价格/按钮）
```

### 6.2 全站视觉一致性检查清单

**批量生成前必须确认**：
- [ ] 所有SKU使用相同光源角度（45度侧逆光）
- [ ] 所有SKU背景色温一致（5500K微暖）
- [ ] 所有SKU道具材质统一（竹/木/亚麻/薄荷青点缀）
- [ ] 所有SKU比例为4:5竖版（手机优先）
- [ ] 深色款（奥利奥/培根/黑芝麻）背景从暖白渐变到炭灰，但保留薄荷青过渡
- [ ] 节庆款（龙/兔/寿桃）点缀色不超15%，主背景仍为暖白

### 6.3 多SKU网格展示效果预览

当11个SKU在Webflow CMS集合列表中并排展示时：
- **色彩和谐**：暖白+原木为主，薄荷青点缀贯穿，无突兀跳色
- **光影统一**：仿佛同一早晨拍摄的全系列
- **层次清晰**：引流款明亮高饱和，利润款质感细腻，旗舰款庄重温润
- **品牌识别**：即使缩小为缩略图，薄荷青点缀+晨光效果仍可识别为"馍范"风格

## 附录：Stable Diffusion WebUI 参数建议

对于使用Stable Diffusion批量生成的场景：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **Checkpoint** | realisticVisionV51 / food摄影专用模型 | 保持写实食物质感 |
| **Lora** | 可选 food photography lora (权重0.3-0.5) | 增强专业食物摄影感 |
| **Sampler** | DPM++ 2M Karras | 稳定细节 |
| **Steps** | 30-40 | 保证蒸汽/流心细节 |
| **CFG Scale** | 7-8 | 平衡提示词遵循与自由度 |
| **Seed** | 固定基础seed，变化-1生成系列 | 保持系列一致性 |
| **ControlNet** | 如有多角度需求，用深度图固定构图 | 确保11张图透视统一 |
